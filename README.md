# IFJ project 2019

Team project for lecture IFJ (Formal Languages and Compilers).

### Goal

The goal was to write a compiler with input of a fictional language IFJ19 and output to an other fictional three-address code IFJcode19.

### Files

The assignment can be found in **ifj2019.pdf** (cz)

The gained score can be foud in **hodnotenie.txt** (cz)

We had to presentate our project, which can be found in folder **prezentace/** for quick review. (cz)

For more detailed description see **dokumentace/**. (cz)

Test files can be found in **tests/**.

### Compiling

If you are in the main directory of the project, run `make`.
